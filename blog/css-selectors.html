CSS Selectors - Class vs Id’s

In CSS, selectors are used to target a specific element or range of elements on a web page. When an element has been targeted, a style or set of styles can be applied to the element.

CSS has a wide range of selectors available to customize your project. The two selectors most commonly used are class and ID. Both of these selectors are used to target elements to which the style should be applied.

When you design a web page, it requires applying certain styles and formatting to get the desired look you want. This is where CSS Selectors come into play.

Selectors give the user the ability to target specific elements on a web page that you can apply styles to. CSS has various types of selectors available such as, universal selectors, descendant selectors, child selectors, and grouping selectors. This blog will just focus on class selectors and id selectors.

The two selectors, class and id, are used to apply styles to elements based on the class and ID assigned to an HTML element.

Class Selector - What is a Class Selector?

A class is a selector is used to select all of the elements with the specific class name. The selector must start with a period (.) and the class name. I have defined my specific class name as “blog-post__card”. By using the class selector, the browser will look for all tags in the page that have a class attribute containing that class name.

Why would you use a class selector?

You would use a class selector when you wanted to select one or multiple HTML elements and put that specific style onto those elements.

ID Selector - What is an ID Selector?

An Id selector is a way to select only the element with a specific id and apply styles to that one unique element. The selector must start with a hash (#) sign.

Why would you use an ID Selector?

You would use an Id selector to specify a unique element. Having an element with a Id selector is only used to identify one single element in our HTML. IDs are only used when one element on the page should have a particular style applied to it.

Summarizing Class vs ID

When deciding what class id you need to use, think about what elements you are trying to apply styles too. If for example, if you’re like me, I have made multiple cards on my home page that I want to be the exact same and have the same styles, you would use the class id. If you have a specific element you want to style which is unique, you would use the Id selector. The difference between CSS class and ID is that CSS class can apply a style to multiple elements. ID can only apply a style to one unique element.
